# 🚀 AI CV Analyzer

An AI-powered CV Analyzer that helps users analyze resumes for jobs and internships using Artificial Intelligence.

This project compares a CV with a Job Description and provides intelligent insights such as match percentage, skill analysis, and interview questions.

---

# ✨ Features

- 📄 Upload Resume / CV
- 📝 Add Job Description
- 📊 CV Match Score Analysis
- 🤖 AI-generated Interview Questions
- 📥 Download Report in DOC Format
- ⚡ Fast and Easy to Use
- 🎯 Internship & Job Fit Analysis

---

# 🛠️ Technologies Used

- Python
- Streamlit / Flask
- NLP (Natural Language Processing)
- Machine Learning
- OpenAI API / Gemini API

---

# 📌 How It Works

1. Upload your CV
2. Paste the Job Description
3. AI analyzes the resume
4. Get:
   - Match Percentage
   - Skill Analysis
   - Suggested Improvements
   - Interview Questions
5. Download the generated report

---

# 💡 Purpose of This Project

The main goal of this project is to help students, fresh graduates, and job seekers understand how well their resume matches a specific role and prepare better for interviews.
